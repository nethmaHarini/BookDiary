## BookDiary v1.0.0 - Initial Release

### What's included
- Splash screen with animated logo
- Register with BCrypt password hashing
- Login with BCrypt verification
- Google Sign-In via Credential Manager + Firebase
- Forgot Password & Reset Password
- Persistent session management
- Profile screen with avatar, stats, edit profile
- Change profile picture (camera / gallery)
- Notification settings (Reading Reminder, Daily Quote, Recommendations) with WorkManager
- Theme preference: Light / Dark / Follow Device
- Accent colour picker: Ocean Blue, Royal Purple, Emerald, Sunset
- Home, Search, Add, Diary tabs (UI designed, logic coming soon)

### Installation
Download **BookDiary-v1.0.0.apk**, enable "Install from unknown sources" on your Android device, and install.
